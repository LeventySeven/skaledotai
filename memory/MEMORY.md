# Project Memory

## Stack
Next.js 16 App Router, React 19, tRPC v11, Drizzle ORM, Better Auth, Zod, SuperJSON, Shadcn UI

## Key Paths
- Routers: `src/server/trpc/routers/`
- Services: `src/server/services/`
- Validations: `src/lib/validations/`
- Constants: `src/lib/constants.ts`
- tRPC client: `src/lib/trpc/client.tsx`
- tRPC provider: `src/lib/trpc/react.tsx`
- Server caller: `src/lib/trpc/server.ts`
- DB schema: `src/db/schema.ts`
